# scamshield
A scam detection tool built in C and Python with a web interface
Lately, I’ve noticed that scam advertisements are becoming increasingly common on social media. As the internet reaches more people, AI-generated text and images are making fraud easier and harder to detect — and data breaches only make things worse. I built ScamShield to help people identify suspicious content before it’s too late.
tech stack:C,Python,HTML/CSS/JS
